
# Visualizador mallas

Este programa renderiza mallas resultantes de análisis de elementos finitos, proporcionando una visualización interactiva de los modelos estructurales y sus resultados.
## Instalación

### Requisitos
* **Python 3.13** o superior
* Gestor de paquetes de Python (pip)
*(Opcional: crear un entorno virtual con venv o conda para aislar las dependencias.)*

### Descarga
```bash
git clone https://github.com/FMM28/Visualizador-3D-MSH
```
Si no tienes git, puedes descargar el proyecto manualmente desde GitHub:

    1. Entra al repositorio.

    2. Haz clic en el botón Code.

    3. Selecciona Download ZIP y descomprime el archivo.

### Dependencias
Una vez dentro del directorio del proyecto, instala las dependencias necesarias ejecutando:
```bash
pip install -r requirements.txt
```
## Ejecución

Para iniciar el programa, abre una terminal dentro del directorio del proyecto y ejecuta el siguiente comando:

```bash
python main.py
```
o si la version instalada de python lo requiere:
```bash
python3 main.py
```
*Nota: asegúrate de haber instalado previamente todas las dependencias antes de ejecutar el programa.*
